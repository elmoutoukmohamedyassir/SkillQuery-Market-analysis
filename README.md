🚀 SkillQuery: 2026 Tech Market Intelligence
Project Goal: Comparing global AI/Data Science trends against the Moroccan job market to identify the "Highest Value" skills for junior developers.

📊 Data Sources
Global: Kaggle Dataset (50,000+ records, 2020-2026).

Local: Curated job postings from Moroccan platforms (Rekrute, LinkedIn).

🛠️ Tech Stack
Language: Python

Libraries: Pandas, Matplotlib, Seaborn.

Analysis: Skill frequency, Salary correlation, Experience-to-Role mapping.
