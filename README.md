# 🚀 SkillQuery+ | 2026 AI Market Intelligence Engine

**SkillQuery+** is a predictive career intelligence platform designed to bridge the gap between global tech trends and the local Moroccan market. Built during my Data Science studies, it transforms 50,000+ data points into actionable career roadmaps.

## 🌟 Core Features

- **🔮 Smart Salary Predictor:** A Random Forest Regressor that predicts market value based on role, location, and seniority.
- **🧬 Job DNA Clustering:** Uses K-Means Unsupervised Learning to group roles by similar skill and salary profiles.
- **🎯 Reactive Skill Gap Analyzer:** Real-time comparison between user skillsets and 2026 market requirements.
- **🇲🇦 Moroccan Tech Hubs:** A specialized analysis of job density across Moroccan cities (Casablanca, Tangier, Rabat) shown in market-share percentages.
- **📋 Career Roadmap Export:** Generate and download a personalized strategic plan as a professional report.

## 🧠 The Tech Stack

- **Backend Logic:** Python (Pandas, NumPy)
- **Machine Learning:** Scikit-Learn (RandomForestRegressor, KMeans Clustering, LabelEncoding)
- **Data Engineering:** Automated Imputation & Feature Engineering (Handling NaNs and Level Standardization)
- **Frontend Dashboard:** Streamlit (Reactive UI)
- **Visualizations:** Plotly Express (Interactive Charts)
